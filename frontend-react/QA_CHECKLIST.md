# Visual and Interaction Checklist

## Key Pages
- `/`
- `/chat`
- `/conversations`
- `/construct/*`
- `/share/[token]`
- `/mobile/chat`

## Visual Checks
- Theme switch: light/dark color token consistency
- Layout: sidebar width, header height, panel spacing
- Typography: title/body text size and weight
- Components: button/input/card/border radius consistency

## Interaction Checks
- Message send and stream rendering cadence
- Conversation switch and list state
- Scroll follow behavior when new chunks arrive
- Loading and error fallback text visibility
